# Data

This folder contains the electricity price data used for the DK1 day-ahead price surface visualisation.

## Source

Data retrieved from the **ENTSO-E Transparency Platform**
(<https://transparency.entsoe.eu>).

| | |
|---|---|
| **Series** | Day-ahead prices |
| **Bidding zone** | DK1 — Western Denmark |
| **Period** | 2022-12-31 to 2026-07-15 |
| **Resolution** | Hourly or 15-minute |
| **Currency / unit** | EUR/MWh |
| **Retrieved on** | 2026-07-15 |
